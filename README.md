# User Management Application

This project is a User Management Application that consists of three main pages:

1. **Login Page**
2. **Registration Page**
3. **Home Page**

## Overview

The User Management Application allows users to register, log in, and manage a list of users. On the Home Page, users can perform CRUD (Create, Read, Update, Delete) operations on user data.

## Features

- **Login Page**: Users can log in to access their account.
- **Registration Page**: New users can create an account by providing their details.
- **Home Page**:
  - Displays a list of all registered users.
  - Each user entry includes buttons to perform the following operations:
    - **Get**: View user details.
    - **Edit**: Update user information.
    - **Delete**: Remove a user from the list.

## Technologies Used

- **Frontend**: React, Redux Toolkit, Axios (for API calls)
- **Backend**: Node.js, Express, MongoDB (for database management)
- **Styling**: CSS Modules
